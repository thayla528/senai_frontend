Perímetro

O perímetro é o conceito mais simples da geometria: ele representa a soma de todos os lados de uma figura.
Se a área é o espaço de dentro, o perímetro é o comprimento do contorno (como se fosse a cerca de um terreno).

Área

O quadrado é a figura mais simples para calcular a área,
pois todos os seus lados são iguais e formam ângulos retos.
A área representa todo o espaço interno da figura

Quadrado
Perímetro

{% if pl is defined %} 4 * {{ n1 }} = {{ pl }} {% endif %}

Perímetro

O perímetro do quadrado é a medida de todo o seu contorno.
Como um quadrado tem os quatro lados exatamente iguais,
o cálculo é o mais simples da geometria.

O perímetro do hexágono é a soma das medidas de todos os seus seis lados. Se o
